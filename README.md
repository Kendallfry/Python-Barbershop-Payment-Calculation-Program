# Python-Barbershop-Payment-Calculation-Program
This is a program I created in Python for a final project for my Principles of Software Development course. It acts as a barborshop tracking app. 
This program has stored values, put in lists, for services provided and prices for a barbershop tracking app. The app has a set number of 6 barbers and stylist. A loop is used for each barber that asks the number of services they provided for a given week. The user of the app, intended for an owner, will be asked how many of each service each employee completed for the week. When the number of each service is input, it is subsequently calculated by the given service price. Totals for all services are then added, and half of the total generated is subtracted as payment to the owner. If half the amount generated is at least $100, then that is what the employee keeps as revenue generated for the week. $100 is the minumum amount of money that goes to the owner weekly for booth rental. Therefore, if an emploee does not generatee at least $100 for the owner, after the revenue split, then they would be on the hook for whatever amount is needed for $100 weekly booth payment. A promt for tips asks how much an emploee was tipped during the week. All tips go directly to the emploee and are added to their weekly 50% profit cut.
